# SUST Online Judge

See also: [SUSTOJ-worker](https://github.com/OpenSUST/sustoj-worker)

## Usage

### Build

```bash
git clone https://github.com/OpenSUST/sustoj.git

cd sustoj

npm install --production

npm run build
```

### Run

```bash
npm start
```
